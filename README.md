# 🌌 Proposal Website
> *Somewhere between nightfall & a wish.*

A cinematic, interactive scroll-based storytelling website designed to present a poetic narrative experience. Instead of a simple question, this project uses smooth animations, hidden interactions, and quiet scene transitions to create an immersive, emotional atmosphere.

**[✨ View the Live Experience](https://proposal-website-drab.vercel.app/)**

## ✨ Features
* **Scroll-Based Storytelling:** A narrative that unfolds gently as the user scrolls.
* **Interactive Elements:** Hidden moments that reveal themselves when hovered or tapped.
* **Cinematic Atmosphere:** Dark mode aesthetics with animated particle effects and glowing text.
* **Smooth Animations:** Powered by Framer Motion for seamless scene transitions.
* **Fully Responsive:** Looks beautiful on both desktop and mobile devices.

## 🛠 Tech Stack
* **Framework:** [Next.js](https://nextjs.org/) (React)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Deployment:** [Vercel](https://vercel.com/)
