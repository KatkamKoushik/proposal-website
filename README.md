# Proposal Website

A cinematic storytelling website built with Next.js and animations.

## Overview
This project is an interactive scroll-based storytelling website designed to present a poetic narrative experience.  
It uses smooth animations and scene transitions to create an immersive atmosphere.

## Features
- Scroll-based storytelling scenes
- Animated particle effects
- Framer Motion animations
- Responsive layout
- Deployed with Vercel

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
